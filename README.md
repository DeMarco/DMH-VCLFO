# DMH-VCLFO
Kosmo format Voltage Controlled tripple LFO module

## Features
6 waveforms, all +5V/-5V
  - Square
  - Saw
  - Ramp
  - Random
  - Sine
  - Triangle

3 frequency ranges
  - Fast
  - Super Fast (Audio rate!)
  - Slow

Hard Sync input

Can also do VCO duties!

Excellent V/Oct trackig over 5 octaves

CV Amount knob serves as Fine Frequency Adjustment when no cable is connected to CV input

Adjustable range for Fine knob

## Current consumption:
+12V --> 65mA

-12V --> 60mA

## Dimensions
Height: 20cm

Width: 10cm

Depth: 10cm


## PCB set
- Panel
- Controls
- Main
- Submodule (x3)

Final Gerbers available in "Gerbers" folder

For viewing/customizing in KiCad, I recommend downloading and installing the Nulshock font from Typodermic: https://typodermicfonts.com/nulshock/
